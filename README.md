# Online Shopping System

Online Shopping System is a DBMS project with both admin and user layouts.

# Installation

1. Install XAMPP or WAMPP.

2. Open XAMPP Control Panel and start **Apache** and **MySQL**.

3. Download the project or clone it using Git:
    
    ```bash
    cd C:\xampp\htdocs\
    git clone https://github.com/raj60221023/online-shopping-system-master.git
    ```
    (Note: Replace the URL with your actual repository URL if different).
    
4. Extract files in `C:\xampp\htdocs\` (if downloaded as zip).

5. Open `http://localhost/phpmyadmin` in your browser.

6. Click on **New** in the side navbar.

7. Give a database name as `onlineshop` and click on **Create**.

8. Select the created database and click on **Import**.

9. Browse the file in the directory `database/onlineshop.sql`.

10. Click **Go** to import the database.

11. Open any browser and type `http://localhost/online-shopping-system-master` (or whatever your folder name is).

12. Register and then login.

13. **Admin Login Details:**
    * Email: `admin@gmail.com` OR Username: `admin`
    * Password: `123456789`
